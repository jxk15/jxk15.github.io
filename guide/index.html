数字电路¶

汇编¶

由于 x86 沉重的历史包袱，系统地学习 x86 汇编并不容易。 如果你选择了 RISC-V 或 MIPS，那么你将少很多烦恼。 汇编指令集与cpu指令集是什么关系？、一文看懂arm架构和x86架构有什么区别

在线编译器：https://godbolt.org。对于 GCC 编译器，使用编译命令 -masm=att 或 -masm=intel 指定生成 ATT 或 Intel 格式的汇编代码。

参考书目（优先级从高到低）：汇编语言 基于 x86 处理器、汇编语言的编程艺术、汇编程序设计与计算机体系结构：软件工程师教程

RISC 精简指令集（reduced instruction set computer）：RISC-V、MIPS、ARM CISC 复杂指令集（complex instruction set computer）：x86

计算机组成：结构化方法¶

推荐本书前四章，后面内容有些过时/不如其他书。

第三章是数字逻辑：讲了 ALU、触发器、寄存器、内存等的门电路实现。

第四章详细讲了一个用微程序控制器实现的CPU，讲了每个时钟周期发生的事情。并从单周期优化到到流水线。但流水线部分不够详细。
